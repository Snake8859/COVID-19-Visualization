# 疫情时空大数据的可视化-前端项目

本项目为基于疫情时空大数据的可视化和辅助决策系统-前端部分，其效果可访问：[传送门](http://www.snake8859.top/DataBoard/index.html)

> 前端技术：
>
> vue+mapbox+echarts
>
> 数据服务：
>
> 区域热力图数据，可参考[传送门](https://github.com/Snake8859/heatmap-snake)
>
> 疫情数据服务，可参考[传送门](https://github.com/Snake8859/COVID-19-Crawler)
